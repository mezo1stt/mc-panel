التعديلات:
1) شكل نيون كامل للواجهة + خلفية جريد وجلو.
2) جزء Update داخل كل سيرفر.
3) ترفع ملف وتدوس Update.
4) السيرفر يخزن الملف داخل uploads/ ويبعث أمر update للسيرفر المقصود.

مهم:
الواجهة والسيرفر جاهزين، لكن تنفيذ التحديث فعليًا داخل Minecraft Plugin لازم تضيفه في البلوجن عندك
بحيث لما /plugin/poll يرجّع:
{
  "type":"update",
  "fileName":"...",
  "storedName":"...",
  "filePath":"..."
}
البلوجن يعرف يعمل إيه بالملف.

التشغيل:
npm install
npm start
